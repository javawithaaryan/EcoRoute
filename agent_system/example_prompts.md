# EcoRoute - Example User Prompts

These example prompts demonstrate how users query EcoRoute across different locations.

---

## 1. Campus Context (Multi-layer Plastic)
```text
I am a student on Prototype Campus. I have an empty CrunchTime chips packet. It has a metallic silver lining inside. How should I dispose of this?
```

---

## 2. Municipal Context (Standard Recyclable)
```text
I have a clean plastic Coca-Cola bottle. I live in Prototype Municipal Zone. Where should this go?
```

---

## 3. Brand Take-Back Context (Shampoo Sachet)
```text
I have empty PureGlow shampoo sachets. I washed them out. I am in Prototype Municipal Zone. Is there a take-back program?
```
